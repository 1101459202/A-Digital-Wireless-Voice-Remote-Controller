# A-Digital-Wireless-Voice-Remote-Controller
一款433数传的无线语音遥控器

硬件配置

MCU: STM32F103RC

无线数传：E62-T100S2 433无线数传模块（UART）

RTC & ENC：RCV8564 SMEC89SP

显示屏：1.12寸 96*96 OLED（SPI）

语音播报：MY1680-16S（UART）

电池：4049110 111x49x4mm 3800mAh聚合物锂电池

按键数：3*5=15 个按键
